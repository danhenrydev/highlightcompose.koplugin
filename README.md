# Integrated highlight & note bar

Note: This plugin is 100% vibe coded for my specific use case, you probably shouldn't use it. 

KOReader plugin that replaces the usual highlight popup with one panel: pick color and style, trim or grow the selection, add or edit a note, open the classic actions (translate, Wikipedia, …) via **…**, then tap **✓** to save.

Tapping an existing highlight opens the same panel (including highlights that already have a note).

Disable the integrated bar under **Main menu → More tools → Integrated highlight & note bar** (or turn off the plugin).
