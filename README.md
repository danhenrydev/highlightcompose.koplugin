# Integrated highlight & note bar

**Note**: This plugin is 100% vibe coded for my specific use case, you probably shouldn't use it. 

KOReader plugin that replaces the usual highlight popup with one compact, **wide** panel: a thin color strip, style symbols + pencil on the next row, then trim/grow and actions. Unless your highlight popup is pinned to top/bottom, the bar is placed **under or above the selection** (not over the text) when there is room, similar to “follow selection” mode.

Tapping an existing highlight opens the same panel (including highlights that already have a note).

Disable the integrated bar under **Main menu → More tools → Integrated highlight & note bar** (or turn off the plugin).
